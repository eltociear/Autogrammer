Autogrammer

Generate syntactically valid code with an LLM. Right in your Browser.

import { autogram } from 'autogrammer' const prompt = ` Write me JSON that captures the following address. Ensure you include the zip code in the JSON output. Address: 1600 Pennsylvania Avenue NW, Washington, DC 20500 ` console.log(await autogram(prompt, { language: 'json' }))
